# 🎨 Assets

This directory contains the official visual assets for Project NONO.

These assets define NONO's visual identity and are referenced throughout the project documentation.

---

# Directory Structure

## official/

Official illustrations and reference artwork.

These images represent the canonical appearance of NONO.

---

## expressions/

Facial expressions and emotional variations.

Examples:

- Smile
- Teasing
- Embarrassed
- Angry
- Surprised

---

## logo/

Official project logos.

---

## banner/

Repository banners and social preview images.

---

## live2d/

Assets prepared for Live2D implementation.

---

# Naming Convention

Use lowercase snake_case.

Examples:

- nono_official.png
- nono_smile.png
- github_banner.png

---

# Design Principle

Official assets should remain visually consistent with the character specifications defined in the documentation.

Visual updates should preserve NONO's core identity.
