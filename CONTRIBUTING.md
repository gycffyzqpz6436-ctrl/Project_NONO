# Contributing to Project NONO

Thank you for your interest in Project NONO.

Project NONO is an open-source project dedicated to designing and implementing a consistent AI character.

The goal is not only to create an AI assistant, but to establish a reproducible digital personality.

---

# Before Contributing

Please read the following documents before making changes.

- README.md
- docs/character/
- docs/implementation/

Understanding NONO's identity and design philosophy is essential before contributing.

---

# Types of Contributions

Contributions are welcome in areas such as:

- Documentation improvements
- Prompt engineering
- Implementation ideas
- Bug reports
- Feature proposals
- Visual assets
- Application development

---

# Pull Requests

Please ensure that your pull request:

- has a clear objective
- follows the existing project structure
- preserves NONO's personality and identity
- includes appropriate documentation updates when necessary

---

# Commit Messages

Follow Conventional Commits where possible.

Examples:

- feat:
- docs:
- fix:
- refactor:
- chore:

---

# Respect the Character

NONO's identity is the heart of this project.

Changes that significantly alter her personality or design should be discussed before implementation.

Consistency is more important than adding new features.

---

# Thank You

Every contribution helps Project NONO grow.

Thank you for helping build NONO.
